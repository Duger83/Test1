# Репозиторий домашних заданий по курсу Python (Data Science)

Данный репозиторий содержит файлы с решениями домашних заданий по следующим темам:
- Основы Python (ООП, декораторы, итераторы, функциональное программирование)
- Библиотеки для анализа данных: NumPy, Pandas
- Визуализация данных (Matplotlib)
- Машинное обучение (Scikit-learn): Линейная регрессия, Деревья решений, Случайный лес, Градиентный бустинг, Кластеризация
- Базы данных и SQL
- Другие темы (многопоточность, тестирование, веб-скрапинг)

## Структура репозитория

В корне репозитория расположены скрипты `.py` и ноутбуки `.ipynb`, сгруппированные по темам. Названия файлов отражают тему задания (например, `Ex1_LR.py` — Линейная регрессия, `Ex1_kMeans.py` — Кластеризация).

**Основные категории файлов:**
- `Ex1_LR.py`, `Ex1_PR.py` — Задачи по линейной и полиномиальной регрессии.
- `Ex1_DT.py`, `Ex1_RF.py`, `Ex1_GB.py` — Задачи по деревьям решений, случайному лесу и градиентному бустингу.
- `Ex1_kMeans.py`, `Ex1_PCA.py` — Кластеризация и понижение размерности.
- `Ex1_Queries.py`, `Ex2_PySpark.py` — Работа с SQL и PySpark.
- `Iterators_Ex*.py`, `Functions_Ex*.py` — Задачи по итераторам, генераторам и функциональному программированию.
- `chart_ex*.py` — Визуализация данных.
- `BikeStore_dump_sql` — Дамп базы данных.
- `*.csv`, `*.txt` — Наборы данных, использованные в работах.
